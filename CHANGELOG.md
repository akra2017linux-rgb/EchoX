## [1.0.0] - 2024-07-21

### Added
- Initial release of EchoX
- Modern dark UI inspired by Spotify
- Smart folder management with deep MP3 scanning
- Real-time music search functionality
- Shuffle and repeat playback modes
- Keyboard shortcuts (F5-F8) for quick control
- Album artwork extraction from ID3 tags
- Persistent music library and settings
- Cross-fade between tracks
- Volume control with smooth adjustments
- Keyboard shortcuts documentation

### Features
- Built with CustomTkinter for modern UI
- Pygame audio engine for reliable playback
- Mutagen for ID3 tag reading and metadata extraction
- PyInstaller integration for Windows executables
- Inno Setup installer creation
- Settings persistence to local configuration
